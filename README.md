# Organ-Donation-Management-System
It helps to keep record of people who wants to donate and who wants any organ also an admin window who can approve the requests.

## Project Description

The Organ Donation Management System is a web-based application designed to manage and streamline the process of organ donation.
It helps connect donors, recipients, and administrators on a single platform, ensuring transparency, proper record management, and efficient organ allocation.

This system aims to reduce manual work, avoid data duplication, and save time while handling sensitive organ donation data securely.

## Objectives of the Project

* To maintain a centralized database of organ donors and recipients
* To simplify the organ donation registration process
* To manage organ requests and approvals in a structured manner
* To ensure secure and reliable data handling

## Users of the System

1. **Admin**

   * Manages users and organ requests
   * Verifies donor and recipient details
   * Maintains system records

2. **Donor**

   * Registers personal and medical details
   * Selects organs for donation
   * Updates availability status

3. **Recipient**

   * Requests required organ
   * Views request status

4. **Hospital**

   * Reviews donor and recipient data
   * Coordinates transplant processes

## Features

* User Registration and Login
* Donor and Recipient Management
* Organ Availability Tracking
* Request and Approval System
* Secure Data Storage
* User-friendly Interface

## Technologies Used

* **Frontend:** HTML, CSS
* **Backend:** Pyhton flask
* **Database:** SQL lite
